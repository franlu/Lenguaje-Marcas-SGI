# Actividad 00 - Selectores básicos

**Objetivo:** Aprender a aplicar estilos básicos utilizando selectores de etiquetas, clases e ID en una estructura HTML5.

Aplicar los siguientes estilos:

- Cambia el color del texto del h1 con el id titulo-principal a azul oscuro.
- Haz que todos los h2 tengan un subrayado y un color de texto gris.
- Aplica un fondo de color gris claro a los p que tengan la clase descripcion.
- Cambia el color del texto de todos los elementos de la lista (li) con la clase servicio a verde.
- Estiliza el footer para que tenga un fondo de color negro y el texto en blanco.
- Haz que el texto de los enlaces (a) en la nav tenga un color azul y que cambie a rojo cuando se pase el ratón por encima (:hover).
- Aplica estilos para que los elementos ul tengan un margen a la izquierda de 20px y un fondo de color amarillo pálido, y que el nav tenga un borde inferior de 2px de color gris oscuro para que resalte la barra de navegación.

# Actividad 000 - Selectores básicos

- Cambia el color del texto del h1 con el id encabezado-principal a rojo oscuro.

- Haz que todos los h3 tengan un color de texto morado y un estilo de texto en negrita.

- Aplica un fondo beige a todos los párrafos (p) que tengan la clase detalle.

- Configura los elementos de la lista (li) con la clase item-importante para que su texto sea azul marino.

- Estiliza el header para que tenga un fondo verde oscuro y un color de texto blanco.

- Haz que los enlaces (a) dentro del footer tengan un color gris oscuro y cambien a naranja al pasar el ratón por encima usando :hover.

- Aplica estilos para que todos los elementos ol tengan un borde izquierdo de 3px sólido en color granate y un padding-left de 15px.

- Haz que la sección con la clase contenedor tenga un fondo celeste y un borde redondeado (border-radius).

- Configura que todos los span dentro de un div con clase alerta aparezcan con texto en rojo y en cursiva.

- Establece un borde superior de 2px en color verde para el elemento main y un margen superior de 20px.

- Haz que los botones (button) cambien su color de fondo a gris cuando el ratón pase por encima (:hover).

- Aplica un color marrón a todos los elementos strong que estén dentro de un párrafo.

- Configura que las imágenes (img) dentro de un article tengan un borde negro de 1px.

- Haz que los elementos section tengan un padding de 10px y un fondo de color arena clara.

- Define que todos los elementos con la clase nota tengan un color de texto vino tinto y un tamaño de letra de 18px.
